# CODSOFT Machine Learning Tasks

This repository contains a collection of machine learning projects completed as part of internship-style tasks. Each project focuses on solving a real-world problem using practical ML workflows.

The emphasis is on understanding the full pipeline — from data preprocessing to model evaluation and tuning.

---

## 📌 Overview

These projects cover:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Model building and evaluation
* Hyperparameter tuning

All work is implemented using Jupyter Notebooks for clarity and step-by-step learning.

---

## 📂 Projects Included

### 1. Movie Genre Classification

A model that predicts the genre of a movie based on input features such as text or metadata.

**Focus areas:**

* Text preprocessing
* Feature extraction
* Classification models

---

### 2. Customer Churn Prediction

A predictive model to identify whether a customer is likely to leave a service.

**Focus areas:**

* Handling imbalanced data
* Feature importance analysis
* Hyperparameter tuning (GridSearchCV)

---

### 3. Credit Card Fraud Detection

A machine learning model designed to detect fraudulent financial transactions based on customer and transaction data.

**Key Features:**

* Feature engineering using transaction time, customer age, and geographic distance
* Distance calculation between merchant and customer using geolocation
* Handling severe class imbalance using SMOTE
* End-to-end pipeline combining preprocessing and modeling

**Modeling Approach:**

* Random Forest Classifier with class balancing
* Imbalanced-learn pipeline (`SMOTE + Random Forest`)
* Evaluation using accuracy, confusion matrix, and ROC-AUC

**Focus areas:**

* Imbalanced classification problems
* Real-world feature engineering
* Pipeline-based ML workflows


## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Krish14-ai/CODSOFT.git
   cd CODSOFT
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Launch notebooks:

   ```bash
   jupyter notebook
   ```
